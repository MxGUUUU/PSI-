    # PSI Project

    ## Overview
    This project implements the **Ψ‑Codex**, a recursive identity model that blends
    symbolic logic, bifurcation theory, and high‑dimensional memory lattices
    (Z₄ ⊗ E₈).  Drawing from catastrophe theory, topological quantum mechanics,
    and cognitive neuroscience, it treats *identity* as a phase‑dependent
    attractor in a stress‑modulated manifold.
    Python‑based simulations analyse coherence thresholds,
    entropy / stress‑energy metrics, and shadow‑integration processes.

    ## Installation
    To get started, create a virtual environment and install requirements:

    ```bash
    pip install -r requirements.txt
    ```

    If you plan to use an OpenAI‑Codex–related tool for integration:

    ```bash
    npm install -g @openai/codex
    ```

    ## Usage
    Run the main Ψ‑Codex simulation and generate reports:

    ```bash
    python -m psi_codex.simulator
    ```

    This produces plots plus **Psi_Codex_Recursive_Identity.pdf**
    in the project root.

    ## Testing
    Ensure *pytest* and *PyPDF2* are available, then:

    ```bash
    pytest
    ```
